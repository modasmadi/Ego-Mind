# Million Logo Page (Arabic Edition)

A modern interactive reinterpretation of the classic "Million Dollar Homepage",
built with Arabic-first UX, smooth zooming, draggable grid, and Firebase backend.

## Features
- Infinite grid with zoom & drag
- Arabic (RTL) optimized UI
- User cells with image, text, likes
- Firebase Firestore backend
- Mobile friendly

## How to Use
1. Drag to explore
2. Zoom to find a cell
3. Double click a cell to view details
4. Reserve your own cell

## Deployment
- Works on GitHub Pages
- Firebase keys are client-side (lock rules before production)
